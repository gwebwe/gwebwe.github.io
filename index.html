Iot To Improve Modern Day life

I worked in a team of four that created a home system that allowed a user to monitor and control their home appliances through a website. There were two main modules to the project, a local controller and a website. The local controller would get communicate with the website to get instructions and would send signals to the home appliances to turn them on or off.

I was responsible for designing the website that would communicate with the local controller. I used a LAMP stack to create the website and deployed the website using Amazon Web Services EC2. We also were responsible for documentation of the project, which included project overview, requirements, design, testing, results, and conclusion.
